# Teste Power BI - League of Legends

Projeto de análise de dados de League of Legends usando Power BI.

## Descrição

Dashboard e análises de dados do jogo League of Legends criado em Power BI para visualização de estatísticas e métricas.

## Conteúdo

- `Teste LoL.pbix` - Arquivo do projeto Power BI
- `ezgif-2-b0fc567224a3-1.webp` - Preview do dashboard

## Tecnologias

- Power BI
- Business Intelligence
- Data Visualization
- DAX (Data Analysis Expressions)

## Como Visualizar

Abra o arquivo `.pbix` no Power BI Desktop.
